# BetterBrightness

In vanilla Minecraft, brightness ranges from 0 to 100. With BetterBrightness, the brightness range is now 0–10, where level 1 matches Minecraft's old maximum (100), and level 10 goes up to ten times brighter.

## Requirements

* LeviLauncher — <https://github.com/LiteLDev/LeviLaunchroid>
* libBetterBrightness mod installed

## Installation

1. Download libBetterBrightness from [Releases](https://github.com/OpenMCBE/LeviLauncher-BetterBrightness/releases) or [GitHub Actions](https://github.com/OpenMCBE/LeviLauncher-BetterBrightness/actions)
2. Install LeviLauncher
3. Add libBetterBrightness mod in LeviLauncher
4. Launch Minecraft through LeviLauncher
